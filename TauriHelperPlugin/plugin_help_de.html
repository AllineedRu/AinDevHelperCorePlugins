Tauri Helper Plugin - Dokumentation

Über das Plugin

Ein Plugin, das den Entwickler bei verschiedenen Aktivitäten im Zusammenhang mit der Entwicklung von Anwendungen auf Tauri unterstützt.

Unterstützte Plugin-Aktionen

Das Plugin unterstützt die folgenden Aktionen:

  • Erstellen Sie eine neue Tauri-Anwendung
  • Öffnen Sie die Tauri-Anwendung in Visual Studio Code
  • Browser – Öffnen Sie die Hauptseite der Tauri-Website
  • Browser – Öffnen Sie die Seite „Guides“ der Tauri-Website
  • Browser – Öffnen Sie die Seite „Anleitungen → Erste Schritte → Voraussetzungen“ auf der Tauri-Website
  • Browser – Öffnen Sie die Seite „Anleitungen → Erste Schritte → Schnellstart“ auf der Tauri-Website
  • Browser – Öffnen Sie die Seite „Anleitungen → Entwicklung → Entwicklungszyklus“ der Tauri-Website
  • Browser – Öffnen Sie die Seite „API → Konfiguration“ der Tauri-Website
  • Browser – Öffnen Sie die Seite „API → CLI“ der Tauri-Website
  • Browser – Öffnen Sie die Seite „API → JavaScript / TypeScript“ der Tauri-Website

Beschreibung der Aktionen

Nachfolgend finden Sie eine Beschreibung, wie die vom Plugin unterstützten Aktionen funktionieren.

Erstellen Sie eine neue Tauri-Anwendung

Eine parametrisierte Aktion hilft Ihnen, schnell eine neue Tauri-Anwendung zu erstellen.

Das Plugin führt den Befehl cmd /c npm create tauri-app@latest ${appName} aus und startet den cmd-Prozess mit Argumente npm create tauri-app@latest ${appName} und Schlüssel /c.

Aktionsparameter

  • appName - Name der neuen Tauri-Anwendung
  • appPath - Arbeitsverzeichnis zum Erstellen einer neuen Tauri-Anwendung beim Erstellen und Starten des cmd-Prozesses

Öffnen Sie die Tauri-Anwendung in Visual Studio Code

Die parametrisierte Aktion versucht, die angegebene Tauri-Anwendung in der Visual Studio Code-Umgebung zu öffnen. Wenn die Entwicklungsumgebung Visual Studio Code auf Ihrem Computer installiert ist und das Verzeichnis als Verzeichnis mit einer vorhandenen Tauri-Anwendung angegeben wurde, wird das Projekt mit dieser Anwendung in Visual Studio Code geöffnet.

Wenn eine Aktion ausgeführt wird, führt das Plugin den Befehl code -r ${appPath} aus, startet die Visual Studio Code-Entwicklungsumgebung und übergibt ihr das im Aktionsparameter angegebene Verzeichnis.

Aktionsparameter

  • appPath - Pfad zum vorhandenen Verzeichnis, das die fertige Tauri-Anwendung enthält, um sie in Visual Studio Code zu öffnen
