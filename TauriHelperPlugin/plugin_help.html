Tauri Helper Plugin - Документация

О плагине

Плагин, помогающий разработчику с выполнением различных действий, связанных с разработкой приложений на Tauri.

Поддерживаемые действия плагина

Плагин поддерживает следующие действия:

  • Создать новое Tauri приложение
  • Открыть Tauri приложение в среде Visual Studio Code
  • Браузер - Открыть главную страницу сайта Tauri
  • Браузер - Открыть страницу "Guides" сайта Tauri
  • Браузер - Открыть страницу "Guides → Getting Started → Prerequisites" сайта Tauri
  • Браузер - Открыть страницу "Guides → Getting Started → Quick Start" сайта Tauri
  • Браузер - Открыть страницу "Guides → Development → Development Cycle" сайта Tauri
  • Браузер - Открыть страницу "API → Configuration" сайта Tauri
  • Браузер - Открыть страницу "API → CLI" сайта Tauri
  • Браузер - Открыть страницу "API → JavaScript / TypeScript" сайта Tauri

Описание действий

Ниже приводится описание работы действий, поддерживаемых плагином.

Создать новое Tauri приложение

Параметризованное действие помогает быстро создать новое приложение Tauri.

Плагин выполняет команду cmd /c npm create tauri-app@latest ${appName}, запуская процесс cmd с аргументами npm create tauri-app@latest ${appName} и ключом /c.

Параметры действия

  • appName - название нового Tauri приложения
  • appPath - рабочий каталог для создания нового Tauri приложения при его создании и запуске процесса cmd

Открыть Tauri приложение в среде Visual Studio Code

Параметризованное действие пробует открыть указанное Tauri приложение в среде Visual Studio Code. Если среда разработки Visual Studio Code установлена на компьютере, и в качестве каталога был указан каталог, содержащий существующее приложение Tauri, то проект с этим приложением будет открыт в Visual Studio Code.

При выполнении действия плагин выполняет команду code -r ${appPath}, запуская среду разработки Visual Studio Code и передавая ей каталог, указанный в параметре действия.

Параметры действия

  • appPath - путь к существующему каталогу, содержащему готовое приложение Tauri для его открытия в Visual Studio Code
