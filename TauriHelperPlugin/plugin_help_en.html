Tauri Helper Plugin - Documentation

About the plugin

A plugin that helps the developer with various activities related to developing applications on Tauri.

Supported plugin actions

The plugin supports the following actions:

  • Create a new Tauri application
  • Open the Tauri application in Visual Studio Code
  • Browser - Open the main page of the Tauri website
  • Browser - Open the "Guides" page of the Tauri website
  • Browser - Open the "Guides → Getting Started → Prerequisites" page of the Tauri website
  • Browser - Open the "Guides → Getting Started → Quick Start" page of the Tauri website
  • Browser - Open the page "Guides → Development → Development Cycle" of the Tauri website
  • Browser - Open the "API → Configuration" page of the Tauri website
  • Browser - Open the "API → CLI" page of the Tauri website
  • Browser - Open the "API → JavaScript / TypeScript" page of the Tauri website

Description of actions

Below is a description of how the actions supported by the plugin work.

Create a new Tauri application

A parameterized action helps you quickly create a new Tauri application.

The plugin executes the command cmd /c npm create tauri-app@latest ${appName}, starting the cmd process with arguments npm create tauri-app@latest ${appName} and key /c.

Action Parameters

  • appName - name of the new Tauri application
  • appPath - working directory for creating a new Tauri application when creating it and starting the cmd process

Open the Tauri application in Visual Studio Code

The parameterized action attempts to open the specified Tauri application in the Visual Studio Code environment. If the Visual Studio Code development environment is installed on your computer, and the directory was specified as the directory containing an existing Tauri application, then the project with that application will be opened in Visual Studio Code.

When an action is executed, the plugin executes the command code -r ${appPath}, starting the Visual Studio Code development environment and passing it the directory specified in the action parameter.

Action Parameters

  • appPath - path to the existing directory containing the finished Tauri application for opening it in Visual Studio Code

Browser - Open the main page of the Tauri website

When executing the action, the default browser installed on the system will be opened, after which the action will open the Web page at https://tauri.app/
