React Helper Plugin - Documentation

About the plugin

A plugin that helps the developer with various activities related to developing applications in React.

Supported plugin actions

The plugin supports the following actions:

  • Create a new React application
  • Open React application in Visual Studio Code
  • Browser - Open the main page of the React site
  • Browser - Open the "Quick Start" page of the React site

Description of actions

Below is a description of how the actions supported by the plugin work.

Create a new React application

A parameterized action helps you quickly create a new React application.

The plugin executes the command cmd /c npx create-react-app ${appName}, starting the cmd process with arguments npx create-react-app ${appName} and key /c.

Action Parameters

  • appName - name of the new React application
  • appPath - working directory for creating a new React application when creating it and starting the cmd process

Open the React application in Visual Studio Code

The parameterized action attempts to open the specified React application in the Visual Studio Code environment. If the Visual Studio Code development environment is installed on your computer, and the directory was specified as the directory containing an existing React application, then the project with that application will be opened in Visual Studio Code.

When an action is executed, the plugin executes the command code -r ${appPath}, starting the Visual Studio Code development environment and passing it the directory specified in the action parameter.

Action Parameters

  • appPath - path to the existing directory containing the finished React application for opening it in Visual Studio Code

Browser - Open the main page of the React site

When executing the action, the default browser installed on the system will be opened, after which the action will open the Web page at https://react.dev/

Browser - Open the "Quick Start" page of the React site

When executing the action, the default browser installed on the system will be opened, after which the action will open the Web page at https://react.dev/learn
