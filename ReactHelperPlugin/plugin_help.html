React Helper Plugin - Документация

О плагине

Плагин, помогающий разработчику с выполнением различных действий, связанных с разработкой приложений на React.

Поддерживаемые действия плагина

Плагин поддерживает следующие действия:

  • Создать новое React приложение
  • Открыть React приложение в среде Visual Studio Code
  • Браузер - Открыть главную страницу сайта React
  • Браузер - Открыть страницу "Quick Start" сайта React

Описание действий

Ниже приводится описание работы действий, поддерживаемых плагином.

Создать новое React приложение

Параметризованное действие помогает быстро создать новое приложение React.

Плагин выполняет команду cmd /c npx create-react-app ${appName}, запуская процесс cmd с аргументами npx create-react-app ${appName} и ключом /c.

Параметры действия

  • appName - название нового React приложения
  • appPath - рабочий каталог для создания нового React приложения при его создании и запуске процесса cmd

Открыть React приложение в среде Visual Studio Code

Параметризованное действие пробует открыть указанное React приложение в среде Visual Studio Code. Если среда разработки Visual Studio Code установлена на компьютере, и в качестве каталога был указан каталог, содержащий существующее приложение React, то проект с этим приложением будет открыт в Visual Studio Code.

При выполнении действия плагин выполняет команду code -r ${appPath}, запуская среду разработки Visual Studio Code и передавая ей каталог, указанный в параметре действия.

Параметры действия

  • appPath - путь к существующему каталогу, содержащему готовое приложение React для его открытия в Visual Studio Code

Браузер - Открыть главную страницу сайта React

При выполнении действия будет открыт браузер по умолчанию, установленный в системе, после чего действие откроет Веб-страницу по адресу https://react.dev/

Браузер - Открыть страницу "Quick Start" сайта React

При выполнении действия будет открыт браузер по умолчанию, установленный в системе, после чего действие откроет Веб-страницу по адресу https://react.dev/learn
