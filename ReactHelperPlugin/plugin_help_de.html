React Helper Plugin - Dokumentation

Über das Plugin

Ein Plugin, das den Entwickler bei verschiedenen Aktivitäten im Zusammenhang mit der Entwicklung von Anwendungen in React unterstützt.

Unterstützte Plugin-Aktionen

Das Plugin unterstützt die folgenden Aktionen:

  • Erstellen Sie eine neue React-Anwendung
  • Öffnen Sie die React-Anwendung in Visual Studio Code
  • Browser – Öffnen Sie die Hauptseite der React-Site
  • Browser – Öffnen Sie die Seite „Schnellstart“ der React-Site

Beschreibung der Aktionen

Nachfolgend finden Sie eine Beschreibung, wie die vom Plugin unterstützten Aktionen funktionieren.

Erstellen Sie eine neue React-Anwendung

Eine parametrisierte Aktion hilft Ihnen, schnell eine neue React-Anwendung zu erstellen.

Das Plugin führt den Befehl cmd /c npx create-react-app ${appName} aus und startet den cmd-Prozess mit Argumente npx create-react-app ${appName} und Schlüssel /c.

Aktionsparameter

  • appName - Name der neuen React-Anwendung
  • appPath - Arbeitsverzeichnis zum Erstellen einer neuen React-Anwendung beim Erstellen und Starten des cmd-Prozesses

Öffnen Sie die React-Anwendung in Visual Studio Code

Die parametrisierte Aktion versucht, die angegebene React-Anwendung in der Visual Studio Code-Umgebung zu öffnen. Wenn die Entwicklungsumgebung Visual Studio Code auf Ihrem Computer installiert ist und das Verzeichnis als Verzeichnis mit einer vorhandenen React-Anwendung angegeben wurde, wird das Projekt mit dieser Anwendung in Visual Studio Code geöffnet.

Wenn eine Aktion ausgeführt wird, führt das Plugin den Befehl code -r ${appPath} aus, startet die Visual Studio Code-Entwicklungsumgebung und übergibt ihr das im Aktionsparameter angegebene Verzeichnis.

Aktionsparameter

  • appPath - Pfad zum vorhandenen Verzeichnis, das die fertige React-Anwendung enthält, um sie in Visual Studio Code zu öffnen

Browser – Öffnen Sie die Hauptseite der React-Site

Beim Ausführen der Aktion wird der auf dem System installierte Standardbrowser geöffnet, woraufhin die Aktion die Webseite unter https://react.dev/ öffnet

Browser – Öffnen Sie die Seite „Schnellstart“ der React-Site

Beim Ausführen der Aktion wird der auf dem System installierte Standardbrowser geöffnet, woraufhin die Aktion die Webseite unter https://react.dev/learn öffnet
